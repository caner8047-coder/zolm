# ZOLM WooCommerce Eklentileri

## ZOLM Booster

Kurulacak tek paket:

- `zolm-booster.zip`

Bu paket WooCommerce tarafındaki ZOLM geliştirmelerini modüller halinde taşır:

- **Trendyol Yorumları:** Trendyol yorumlarını WooCommerce ürün sayfalarında widget, rozet ve REST API ile yönetir.
- **WhatsApp Köprüsü:** Sipariş, müşteri, sepet, izin, stok bildirim ve yüzen WhatsApp butonu akışlarını yönetir.

WordPress yönetiminde menü:

- `ZOLM Booster > Modüller`
- `ZOLM Booster > Trendyol Yorumları`
- `ZOLM Booster > WhatsApp`
- `ZOLM Booster > Trendyol Ayarları`

Eski `ZOLM WhatsApp Booster` ayrı eklentisi kullanılmamalıdır. Yeni `ZOLM Booster` paketi eski WhatsApp ayar anahtarlarını okuyarak çalışır; bu nedenle eski eklenti etkisizleştirildikten sonra WhatsApp modülü aynı ZOLM URL, webhook secret ve store ID değerlerini kullanabilir.
