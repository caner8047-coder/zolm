## Canli Sunucu Komut Sirasi

Sunucuda uygulama dizinine gec:

```bash
cd /home/zolmcomt/m.zolm.com.tr
```

Canli DB ve env yedegi al:

```bash
cp .env .env.v06-before-v07-$(date +%F-%H%M)
mysqldump -u DB_USER -p DB_NAME > zolm-before-v07-$(date +%F-%H%M).sql
```

Kod ve env hazir olduktan sonra:

```bash
chmod +x deploy/cpanel/after_upload_v07.sh
deploy/cpanel/after_upload_v07.sh
```

Script kullanilmayacaksa manuel komutlar:

```bash
mysql -u DB_USER -p DB_NAME < database/sql/v06_to_v07_preflight.sql
php artisan migrate --force
mysql -u DB_USER -p DB_NAME < database/sql/v06_to_v07_preflight.sql
php artisan config:clear
php artisan cache:clear
php artisan route:clear
php artisan view:clear
php artisan config:cache
php artisan route:cache
php artisan view:cache
php artisan queue:restart
```

Saglik ve pazaryeri testleri:

```bash
php artisan marketplace:health-check --fail-on-warning
php artisan marketplace:dispatch-due-syncs --type=orders --force
php artisan marketplace:dispatch-due-syncs --type=questions --force
php artisan marketplace:smoke-test STORE_ID --type=orders --hours=24 --preview=2
php artisan marketplace:smoke-test STORE_ID --type=questions --hours=168 --preview=2
```

Scheduler cron:

```bash
* * * * * php /home/zolmcomt/m.zolm.com.tr/artisan schedule:run --no-interaction >> /dev/null 2>&1
```

Queue worker:

```bash
php artisan queue:work --queue=default --sleep=3 --tries=3 --timeout=1800 --memory=512
```
