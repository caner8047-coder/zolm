# Chrome Web Store görselleri

`showcase.html`, ZOLM Kurumsal Açık Panel Sistemiyle hazırlanmış mağaza görsellerinin yeniden üretilebilir kaynağıdır. Ekranlarda temsili ürün ve finans verisi kullanılır; her görselin altında bunun finansal karar olmadığı belirtilir.

Beklenen çıktılar:

- `01-product-decision-1280x800.png`
- `02-radar-actions-1280x800.png`
- `03-seller-profit-1280x800.png`

Görseller 1280×800 PNG olmalı ve `npm run extension:release` kapısından geçmelidir.
